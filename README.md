# Hora do Remédio

Aplicativo mobile desenvolvido em **Flutter** para auxiliar no gerenciamento de medicamentos, com foco em **simplicidade, acessibilidade e usabilidade para idosos**.

---

## Sobre o projeto

O **Hora do Remédio** foi criado para ajudar usuários a organizarem seus medicamentos de forma prática e segura.  
O aplicativo permite cadastrar remédios, definir horários, receber lembretes e acompanhar informações pessoais importantes para o cuidado com a saúde.

A proposta do app é oferecer uma interface clara, intuitiva e acessível, tornando o uso fácil mesmo para pessoas com pouca familiaridade com tecnologia.

---

## Funcionalidades
- Cadastro e autenticação de usuários
- Cadastro de medicamentos
- Edição e exclusão de medicamentos
- Lembretes de horários
- Registro de medicamento tomado
- Desfazer registro de medicamento tomado
- Perfil do usuário
- Armazenamento local
- Sincronização com Firebase
- Interface acessível e adaptada para melhor leitura

---

## Tecnologias utilizadas
- **Flutter**
- **Dart**
- **Firebase Authentication**
- **Cloud Firestore**
- **Shared Preferences**
- **Provider**
- **Flutter Local Notifications**

---

## Objetivo do aplicativo
O principal objetivo do **Hora do Remédio** é ajudar no controle correto dos medicamentos, reduzindo esquecimentos e facilitando a rotina de cuidados com a saúde.

O aplicativo foi pensado especialmente para:

- idosos
- pessoas em tratamento contínuo
- usuários que precisam organizar vários medicamentos por horário
- familiares que auxiliam no acompanhamento de remédios
